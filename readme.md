# run yourself 
requires a fairly new v of **node** and **git**!

fork and **pull**.
```sh
-> git add origin https://github.com/purpleblueslime/willyougooutwithme.git 
-> git pull origin
```

install **deps/dev deps** from [**`package.json`**](./package.json)

## `dev`
```sh
-> next dev
```
view at port **3k**.

-> next build
